- 更新项目版本至v2.0.0

- 完善写作能力分析功能
- 增加写作进度趋势分析

## 主要功能
- AI智能写作建议
- 实时写作辅助
- 智能润色和改写
- 写作素材管理
- 收藏夹功能
- 学习进度跟踪

## 技术栈
- 前端：HTML5, CSS3, JavaScript
- 框架：Vue.js
- AI：OpenAI GPT API
- 构建工具：Vite

## 开发环境
- Node.js >= 14.0.0
- npm >= 6.0.0

## 安装和运行
```bash
npm install
npm run dev
```
